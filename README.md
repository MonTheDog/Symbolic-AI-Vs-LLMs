# Symbolic-AI-Vs-LLMs

Repository for the Symbolic AI vs LLMs project for the Artificial Intelligence Fundamentals course @ UniPi.

The aim of this project is to show, evaluate and compare the quality of solutions given by llms against symbolic AI approaches for reasoning tasks.
